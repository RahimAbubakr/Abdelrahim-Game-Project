# 🐍 Snake Game

## 📌 Description

This is a simple Snake Game developed in C.
The player controls a snake that moves around the screen, eats food, and grows longer.
The goal is to score as many points as possible without colliding with the walls or the snake's own body.

---

## ⚙️ Requirements

* GCC compiler
* Terminal (Linux, macOS, or Windows)

---

## 🛠️ Compilation

Run the following command in the project folder:

```bash
gcc *.c -o snake_maze -I/opt/homebrew/include -L/opt/homebrew/lib -lraylib -framework OpenGL -framework Cocoa -framework IOKit -framework CoreAudio


```

---

## ▶️ Run the Game

After compiling, run:

```bash
./snake_maze
```

On Windows:

```bash
snake.exe
```

---

## 🎮 Controls

* Use keyboard keys ( arrow keys) to move the snake
* Eat food to grow
* Avoid collisions
